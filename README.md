# Teslo API

1. Clonar proyecto

2. Inicializar proyecto

   ```
   yarn install
   ```

3. Clonar `.env.template` y renombrarlo como `.env`

4. Cambiar las variables de entorno por las requeridas

5. Levantar la base de datos

   ```
   docker-compose -d
   ```

6. Ejecutar SEED

   ```
   localhost:3000/api/seed
   ```

7. Levantar la app

   ```
   yarn start:dev
   ```
